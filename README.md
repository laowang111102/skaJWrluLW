# 学生成绩管理系统教师版 python167

## 项目概述

学生成绩管理系统教师版是一款便于教师进行成绩管理和分析的教学辅助工具。系统支持基础的注册、登录及退出功能，界面采用Tkinter模块实现，简洁直观。

## 技术栈

- **后端语言**: Python
- **数据库**: MySQL
- **前端图形界面**: Tkinter
- **数据存储**: Excel

## 功能模块

- **用户管理**
  - 教师注册
  - 教师登录
  - 退出系统

- **成绩管理**
  - 添加学生成绩
  - 修改学生成绩
  - 查询学生成绩
  - 成绩数据写入Excel文件

## 系统角色

- 教师：负责注册、登录系统，管理学生成绩。

## 运行环境

- 操作系统：兼容Windows、Linux等。
- Python版本：建议使用3.x版本。

## 部署步骤

1. 安装Python环境。
2. 安装MySQL数据库，创建对应数据库和表。
3. 克隆或下载项目代码。
4. 安装依赖库，如Tkinter、pymysql等。
5. 运行系统。

## 目录结构

```
- main.py: 程序主入口
- views/: 视图文件夹，包含图形界面相关代码
- models/: 模型文件夹，包含数据库操作相关代码
- utils/: 工具文件夹，包含辅助功能相关代码
- static/: 静态资源文件夹，如图片等
```

## 核心亮点

- **便捷的图形界面**: 使用Tkinter模块，操作简便，易于上手。
- **灵活的成绩管理**: 支持添加、修改、查询成绩，并能将数据写入Excel文件。
- **稳定的数据存储**: 采用MySQL数据库，保证数据稳定性和安全性。
